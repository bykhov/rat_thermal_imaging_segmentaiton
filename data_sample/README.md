# `data_sample/`

A small subset of the released corpus, bundled in the repository so that
reviewers can run the smoke test in [`code/data_sample/smoke_test.py`](../code/data_sample/smoke_test.py)
without first downloading the full archive from Figshare.

## Per-rat layout

```
Rat11/
  CSV/                  raw radiometric temperature matrices (one per frame)
  Mask/                 ground-truth 4-class segmentation masks
  Thermal Imaging/      Jet-colormap renderings used as visual reference
  Ground Truth Mask (color)/   colorized renderings of Mask/ on a black background
  Ground Truth Overlay/        Thermal Imaging blended with Mask/
  Predicted Mask (color)/      colorized model predictions (regenerated by smoke_test.py)
  Predicted Overlay/           Thermal Imaging blended with model predictions (regenerated by smoke_test.py)
```

`Ground Truth Overlay/` and `Ground Truth Mask (color)/` are regenerated by
[`code/data_sample/make_overlay.py`](../code/data_sample/make_overlay.py) (and by [`smoke_test.py`](../code/data_sample/smoke_test.py)); `Predicted Overlay/`
and `Predicted Mask (color)/` are regenerated by [`code/data_sample/smoke_test.py`](../code/data_sample/smoke_test.py).


## File conventions

These follow the *Data Records* section of the paper and the
`data_full/README.md` description; they are restated here for the
bundled sample.

### Temperature matrices (`CSV/Thermal_<N>_CSV.csv`)

- 480 rows x 640 columns of float temperatures in degrees Celsius (~2.1 MB/frame);
- 2-line header: line 1 is the originating file path, line 2 is empty;
- the first data column is a row label (`Frame 1,` on the first data row, `,` on the rest) and must be dropped;
- read with:
  ```python
  df = pd.read_csv(path, skiprows=2, header=None)
  temp = df.iloc[:, 1:].values.astype(np.float32)   # (480, 640) float32
  ```

### Segmentation masks (`Mask/Thermal_<N>.png`)

- single-channel `uint8` PNG with discrete class labels:

  | Value | Class      |
  |-------|------------|
  | 0     | Background |
  | 1     | Head       |
  | 2     | Body       |
  | 3     | Tail       |

- must be read with raw-pixel preservation (standard imread paths may apply colormap conversion); use:
  ```python
  stream = np.fromfile(str(path), np.uint8)
  mask = cv2.imdecode(stream, cv2.IMREAD_UNCHANGED)   # (480, 640) uint8
  ```

### Thermal renderings (`Thermal Imaging/Thermal_<N>.png|jpg`)

uint8 RGB visualization of the temperature matrix via the Jet colormap.
Reference only; analysis code should consume the radiometric CSV.
